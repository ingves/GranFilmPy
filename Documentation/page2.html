Run the simulation and get the resultsÂ¶

Once you’ve configure the parameters of the simulation, you can run it with gf()

Then gf works like a dictionary and each quantity provided by GranFilm is stored either in:

  • gf["name_of_the_quantity"]
  • or can be get through the function gf["name_of_the_quantity"](parameters).
gf["energy"]
Returns:the energies (in eV) on which optical properties are calculated
Return type:array
gf["wavelength"]
Returns:same for wavelength (in nm)
Return type:array
gf["epsilon"](material)
Returns:The complex dielectric function of a material used in the simulation
Return type:array
Parameters:material (string) – the material for which you want the dielectric function
gf["Fresnel"](ref_trans,pol,geometry="Island",fmt="Complex",deg="True")
Returns:

The fresnel coefficient (ratio of E-fields) for reflection or transmission for a given polarisation
Return type:

array
Parameters:
  • ref_trans (string) – ‘r’ for reflection, ‘t’ for transmission
  • pol (string) – ‘p’ for a polarisation in the incident plan, ‘s’ for a perpendicular one, ‘ps’ for unpolarised light
  • geometry (string) – ‘Island’ to get the fresnel amplitude for your configuration (islands on substrate), ‘Flat’ to get it for a flat substrate
  • fmt (string) – The format of the fresnel coefficient: ‘Complex’, ‘Real’,’Imag’,’Amplitude’ or ‘Phase’
  • deg (string) – deg = ‘True’ to use degrees for the ‘Phase’ format, ‘False’ for radians
gf["Intensity_coeff"](ref_trans,pol,geometry="Island")
Returns:

The intensity coefficient (ratio of intensities) for reflection or transmission for a given polarisation
Return type:

array
Parameters:
  • ref_trans (string) – ‘R’ for reflection, ‘T’ for transmission
  • pol (string) – ‘p’ for a polarisation in the incident plan, ‘s’ for a perpendicular one, ‘ps’ for unpolarised light
  • geometry (string) – ‘Island’ to get the intensity coefficient for your configuration (islands on substrate), ‘Flat’ to get it for a flat substrate
gf["Diff_intensity_coeff"](ref_trans,pol)
Returns:

The differential intensity coefficient = (X_islands - X_flat) / X_flat for reflection or transmission for a given polarisation
Return type:

array
Parameters:
  • ref_trans (string) – ‘R’ for reflection, ‘T’ for transmission
  • pol (string) – ‘p’ for a polarisation in the incident plan, ‘s’ for a perpendicular one, ‘ps’ for unpolarised light
gf["Potential"](output='plot',fmt="Real",deg='True',equipotential=False)
Returns:

The scalar potential as defined in the “points_file” parameter
Parameters:
  • output – ‘plot’ to plot the potential, ‘potential_map’ for the 2D matrix of the potential around the particle, ‘error’ to get the average discontuinity error of the potential at the interfaces and ‘data’ for the raw potential matrix where each line is [x,y,z,medium_number,Potential_real,Potential_Imag]

    Note: 'plot' and 'potential_map' work for the "classic" points_file 'error' works for the "surface" points_file 'data' works for any points_file (except 'None')

  • fmt (string) – The format of the output: ‘Complex’, ‘Real’,’Imag’,’Amplitude’ or ‘Phase’
  • deg (string) – deg = ‘True’ to use degrees for the ‘Phase’ format, ‘False’ for radians
  • equipotential (boolean) – True -> add equipotentials to the potential plot
gf["Polarizabilities"](orientation="None", type='Dipole', fmt='Complex', deg='True')
Returns:

The polarizability of one particle supported by the substrate
Return type:

array
Parameters:
  • orientation (string) – ‘Parallel’ / ‘Perpendicular’ for the component of the polarizability parallel / normal to the interface
  • type (string) – ‘Dipole’ / ‘Quadrupole’ for the dipolar / quadrupolar polarizability
  • fmt (string) – The format of the output: ‘Complex’, ‘Real’,’Imag’,’Amplitude’ or ‘Phase’
  • deg (string) – deg = ‘True’ to use degrees for the ‘Phase’ format, ‘False’ for radians
gf["Susceptibilities"](type='Dipole', fmt='Complex', deg='True')
Returns:

The surface susceptibilities of the system
Return type:

array
Parameters:
  • type (string) – ‘gamma’,’beta’,’delta’ or ‘tau’ for each susceptibility
  • fmt (string) – The format of the output: ‘Complex’, ‘Real’,’Imag’,’Amplitude’ or ‘Phase’
  • deg (string) – deg = ‘True’ to use degrees for the ‘Phase’ format, ‘False’ for radians
